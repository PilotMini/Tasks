Параметры функций Калькулятор с параметрами по умолчанию

Создайте программу, которая:

  1. Реализует функцию double calculate(double a, double b, char op = '+'), которая выполняет арифметическую операцию над числами a и b.
  2. Параметр op может принимать значения '+', '-', '*', '/'.
  3. Если операция не указана, выполняется сложение.
  4. В функции main() продемонстрируйте работу функции с разным количеством аргументов.
  5. Обработайте случай деления на ноль (верните 0.0 и выведите сообщение об ошибке).
💡 Подсказка из урока: Параметры со значениями по умолчанию должны идти после обязательных параметров.
Пример вывода: 2 + 3 = 5 10 - 4 = 6 5 * 6 = 30 15 / 3 = 5 15 / 0 = Ошибка: деление на ноль! Результат: 0

Передача по значению и по ссылке Обмен значений

Напишите программу, которая:

  1. Реализует три функции для обмена значениями двух переменных:
    • void swapByValue(int a, int b) — передача по значению
    • void swapByReference(int& a, int& b) — передача по ссылке
    • void swapByPointer(int* a, int* b) — передача через указатели
  2. В функции main() создайте две переменные x = 10, y = 20.
  3. Покажите результат каждой функции до и после вызова.
  4. Объясните, почему только две последние функции реально меняют значения переменных.
💡 Из урока: При передаче по значению функция получает копию, поэтому изменения не влияют на оригинал. Ссылки и указатели позволяют изменять оригинальные переменные.

Константные параметры Безопасный вывод

Создайте программу, которая:

  1. Реализует функцию void printInfo(const std::string& name, const int& age), которая выводит информацию о человеке.
  2. Параметры должны быть константными ссылками.
  3. Попробуйте внутри функции изменить значение параметра и убедитесь, что компилятор выдаёт ошибку.
  4. В функции main() создайте переменные userName = "Alice" и userAge = 25.
  5. Передайте их в функцию и выведите результат.
💡 Из урока: Константные ссылки используются для передачи больших объектов без копирования, гарантируя, что они не будут изменены.
Пример вывода: Имя: Alice, Возраст: 25

Оператор return Поиск минимума и максимума

Напишите программу, которая:

  1. Реализует функцию auto findMinMax(const int arr[], int size, int& min, int& max) -> int, которая:
    • Находит минимальный и максимальный элементы в массиве
    • Возвращает через ссылки min и max найденные значения
    • Возвращает сумму всех элементов массива как результат функции
  2. В функции main() создайте массив из 5 целых чисел, введённых пользователем.
  3. Вызовите функцию и выведите минимум, максимум и сумму.
💡 Из урока: Функция может возвращать одно значение через return, а остальные — через параметры по ссылке.
Пример ввода/вывода: Введите 5 чисел: 3 7 1 9 4 Минимум: 1 Максимум: 9 Сумма: 24

Указатели в параметрах Динамический массив

Создайте программу для работы с динамическим массивом:

  1. Функция int* createArray(int size) — выделяет память под массив и возвращает указатель.
  2. Функция void fillArray(int* arr, int size) — заполняет массив случайными числами.
  3. Функция void printArray(const int* arr, int size) — выводит массив.
  4. Функция void resizeArray(int*& arr, int& size, int newSize) — изменяет размер массива (с сохранением старых элементов).
  5. Функция void deleteArray(int* arr) — освобождает память.
  6. В main() продемонстрируйте работу всех функций.
💡 Из урока: Указатели в параметрах позволяют изменять сами указатели (например, при перевыделении памяти), поэтому используется int*& arr.
