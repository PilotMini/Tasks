Указатели на функции Калькулятор с выбором операции

Создайте программу, которая:

  1. Реализует четыре математические функции: add, subtract, multiply, divide.
  2. Объявите указатель на функцию, который может указывать на любую из этих функций.
  3. В функции main() создайте меню для выбора операции (1-4).
  4. В зависимости от выбора, присвойте указателю соответствующую функцию и выполните вычисления с двумя числами, введёнными пользователем.
  5. Выведите результат на экран.
💡 Синтаксис: int (*operation)(int, int); — указатель на функцию, принимающую два int и возвращающую int.
Пример работы: Введите два числа: 10 5 Выберите операцию: 1 - Сложение 2 - Вычитание 3 - Умножение 4 - Деление Ваш выбор: 1 Результат: 10 + 5 = 15

Callback-функции Обработка массива

Напишите программу, которая:

  1. Реализует функцию void processArray(int* arr, int size, void (*processor)(int&)), которая применяет переданную функцию-обработчик к каждому элементу массива.
  2. Реализуйте несколько функций-обработчиков:
    • increment(int& x) — увеличивает значение на 1
    • square(int& x) — возводит в квадрат
    • negate(int& x) — меняет знак на противоположный
  3. В main() создайте массив из 5 целых чисел, выведите исходный массив, затем последовательно примените каждый обработчик, выводя результат после каждой операции.
💡 Callback (обратный вызов) — это функция, которая передаётся в другую функцию для последующего вызова.
Пример вывода: Исходный массив: 1 2 3 4 5 После increment: 2 3 4 5 6 После square: 4 9 16 25 36 После negate: -4 -9 -16 -25 -36

Массив указателей на функции Меню операций

Создайте программу, которая:

  1. Реализует 6 арифметических операций: сложение, вычитание, умножение, деление, остаток от деления, возведение в степень.
  2. Создайте массив указателей на функции, содержащий все эти операции.
  3. В main() предложите пользователю выбрать операцию (1-6), ввести два числа и вычислить результат, вызывая соответствующую функцию из массива.
  4. Если пользователь выбирает деление или остаток от деления, проверьте, что второе число не равно нулю.
  5. Реализуйте функцию для возведения в степень (целые числа, без использования pow).
💡 Массив указателей: int (*operations[6])(int, int) = {add, sub, mul, div, mod, pow};

Указатель на функцию как параметр Поиск в массиве

Напишите программу, которая:

  1. Реализует функцию int* findFirst(int* arr, int size, bool (*predicate)(int)), которая:
    • Принимает указатель на функцию-предикат, проверяющую условие
    • Возвращает указатель на первый элемент, удовлетворяющий условию
    • Если элемент не найден, возвращает nullptr
  2. Реализуйте несколько функций-предикатов:
    • isEven(int x) — проверяет, чётное ли число
    • isPositive(int x) — проверяет, положительное ли число
    • isGreaterThan10(int x) — проверяет, больше ли число 10
  3. В main() создайте массив и протестируйте поиск с разными предикатами.
💡 Предикат — это функция, возвращающая bool. Используется для проверки условий.
Пример вывода: Массив: 3 7 12 5 8 Первый чётный элемент: 12 (индекс 2) Первый положительный: 3 (индекс 0) Первый больше 10: 12 (индекс 2)

Указатели на функции высшего порядка Генератор математических таблиц

Создайте комплексную программу, которая:

  1. Реализует функцию void generateTable(int start, int end, int (*func)(int)), которая:
    • Выводит таблицу значений функции func для чисел от start до end включительно
    • Формат вывода: X → func(X)
  2. Реализуйте несколько математических функций:
    • int square(int x) — x²
    • int cube(int x) — x³
    • int factorial(int x) — факториал (только для x ≤ 10)
    • int fibonacci(int x) — число Фибоначчи
  3. Создайте массив указателей на эти функции.
  4. В main() предложите пользователю:
    • Выбрать функцию из меню
    • Ввести диапазон (начало и конец)
    • Отобразить сгенерированную таблицу
💡 Функции высшего порядка — функции, которые принимают другие функции как аргументы или возвращают их.
Пример вывода (для функции square, диапазон 1-5): 1 → 1 2 → 4 3 → 9 4 → 16 5 → 25
💡 Важные моменты:
  • Указатели на функции позволяют передавать поведение как параметр (стратегии, callback-механизмы).
  • Синтаксис: тип_возврата (*имя_указателя)(список_параметров).
  • Можно создавать массивы указателей на функции для реализации меню и диспетчеризации вызовов.
  • Указатели на функции активно используются в алгоритмах стандартной библиотеки C++ (например, std::sort с компаратором).
