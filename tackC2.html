Основные понятия об указателях

Операция Синтаксис Описание Пример
Объявление указателя тип* имя; Создание переменной-указателя int* ptr;
Адрес переменной &переменная Получение адреса переменной ptr = &x;
Разыменование *указатель Получение значения по адресу cout << *ptr;
Арифметика указателей ptr + n Смещение на n элементов ptr + 1
Нулевой указатель nullptr Указатель, не указывающий никуда int* ptr = nullptr;
int x = 5; // переменная x хранит значение 5
int* ptr = &x; // указатель ptr хранит адрес переменной x

// В памяти:
// x [ 5 ] по адресу 0x1000
// ptr[0x1000] по адресу 0x2000

cout << x; // 5
cout << &x; // 0x1000
cout << ptr; // 0x1000
cout << *ptr; // 5
← переменная
← указатель
← разыменование

Базовые операции с указателями

Напишите программу, которая:

  1. Объявляет целочисленную переменную number со значением 42
  2. Объявляет указатель на целое число ptr
  3. Присваивает указателю адрес переменной number
  4. Выводит на экран:
    • Значение переменной number
    • Адрес переменной number (в шестнадцатеричном формате)
    • Значение указателя ptr (адрес)
    • Значение, на которое указывает указатель (через разыменование)
    • Адрес самого указателя
  5. Изменяет значение переменной через указатель (присваивает 100)
  6. Выводит новое значение переменной
Подсказка: Для вывода адреса используйте cout << (void*)ptr или форматирование hex.

Указатели и массивы

Работа с массивом через указатели (по материалам metanit.com):

  1. Создайте массив целых чисел arr[] = {5, 10, 15, 20, 25}
  2. Объявите указатель ptr и присвойте ему адрес первого элемента массива
  3. Выведите все элементы массива тремя способами:
    • Через индексацию: arr[i]
    • Через арифметику указателей: *(ptr + i)
    • Через имя массива как указателя: *(arr + i)
  4. Используя указатель, найдите сумму всех элементов массива

Арифметика указателей

Исследуйте особенности арифметики указателей:

  1. Создайте массивы разных типов:
    • int intArr[] = {10, 20, 30, 40}
    • double doubleArr[] = {1.1, 2.2, 3.3, 4.4}
    • char charArr[] = {'A', 'B', 'C', 'D'}
  2. Для каждого массива создайте указатель на первый элемент
  3. Выведите адреса элементов и значения при прибавлении 1, 2 к указателю
  4. Объясните разницу в смещении для разных типов данных
Пример вывода: int: ptr = 0x1000, ptr+1 = 0x1004 (смещение 4 байта) double: ptr = 0x2000, ptr+1 = 0x2008 (смещение 8 байт) char: ptr = 0x3000, ptr+1 = 0x3001 (смещение 1 байт)

Указатели и функции (параметры по указателю)

Реализуйте следующие функции с использованием указателей:

  1. void increment(int* ptr) - увеличивает значение на 1
  2. bool findInArray(int* arr, int size, int target, int** result) - поиск элемента в массиве, возвращает указатель на найденный элемент через параметр

  4. int* createCopy(int* arr, int size) - создает копию массива в динамической памяти

Комплексная задача: динамический массив с функциями

Создайте программу для работы с динамическим массивом:

  1. Функция int* createArray(int size) - создает массив заданного размера
  2. Функция void fillArray(int* arr, int size) - заполняет массив случайными числами
  3. Функция void printArray(int* arr, int size) - выводит массив
  4. Функция int* expandArray(int* arr, int& size, int newSize) - расширяет массив
  5. Функция int* insertElement(int* arr, int& size, int element, int position) - вставляет элемент
  6. Функция int* removeElement(int* arr, int& size, int position) - удаляет элемент
  7. Функция void freeArray(int* arr) - освобождает память
  8. Создайте меню для демонстрации всех операций
Меню программы: 1. Создать массив 2. Заполнить случайными числами 3. Вывести массив 4. Расширить массив 5. Вставить элемент 6. Удалить элемент 7. Выход
