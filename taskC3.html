Базовые математические функции БАЗОВЫЙ

Создайте набор функций для выполнения различных математических операций:

  1. Функция int power(int base, int exponent) - возведение числа в степень (без использования pow)
  2. Функция int factorial(int n) - вычисление факториала числа
  3. Функция bool isPrime(int n) - проверка, является ли число простым
  4. Функция int gcd(int a, int b) - нахождение наибольшего общего делителя (алгоритм Евклида)
  5. Функция int lcm(int a, int b) - нахождение наименьшего общего кратного
  6. В главной функции (main) создайте меню для выбора и тестирования каждой функции
Подсказка: Факториал определен только для неотрицательных целых чисел. Для рекурсивной реализации factorial можете использовать рекурсию.

Работа с массивами через функции СРЕДНИЙ

Реализуйте набор функций для работы с одномерными массивами:

  1. void fillArray(int arr[], int size, int minVal, int maxVal) - заполнение массива случайными числами
  2. void printArray(const int arr[], int size) - вывод массива на экран
  3. int findMax(const int arr[], int size) - поиск максимального элемента
  4. int findMin(const int arr[], int size) - поиск минимального элемента
  5. int findIndex(const int arr[], int size, int value) - поиск индекса элемента по значению
  6. void sortArray(int arr[], int size) - сортировка массива по возрастанию (пузырьковая сортировка)
  7. int calculateSum(const int arr[], int size) - вычисление суммы элементов
  8. double calculateAverage(const int arr[], int size) - вычисление среднего арифметического
  9. void reverseArray(int arr[], int size) - переворот массива
  10. В главной функции создайте массив, продемонстрируйте работу всех функций

Перегрузка функций СРЕДНИЙ

Создайте перегруженные функции для различных типов данных:

  1. Функция maxValue:
    • int maxValue(int a, int b) - возвращает большее из двух целых чисел
    • double maxValue(double a, double b) - возвращает большее из двух вещественных чисел
    • char maxValue(char a, char b) - возвращает больший символ (по ASCII)
  2. Функция print:
    • void print(int value) - выводит целое число
    • void print(double value, int precision = 2) - выводит вещественное число с заданной точностью
    • void print(const char* str) - выводит строку
    • void print(const int arr[], int size) - выводит массив
  3. Функция area:
    • double area(double radius) - площадь круга
    • double area(double length, double width) - площадь прямоугольника
    • double area(double base, double height, bool isTriangle) - площадь треугольника
  4. Продемонстрируйте вызов каждой перегруженной функции

Рекурсивные функции ПРОДВИНУТЫЙ

Реализуйте следующие рекурсивные функции:

  1. int fibonacci(int n) - вычисление n-го числа Фибоначчи (рекурсивно)
  2. int sumDigits(int n) - сумма цифр числа (рекурсивно)
  3. bool isPalindrome(const char* str, int start, int end) - проверка строки на палиндром
  4. void towerOfHanoi(int n, char from, char to, char aux) - решение задачи "Ханойские башни"
  5. int binarySearch(const int arr[], int left, int right, int target) - бинарный поиск (рекурсивный)
  6. void printReverse(const char* str) - вывод строки в обратном порядке (рекурсивно)
  7. int powerRecursive(int base, int exponent) - возведение в степень (рекурсивно)
  8. В главной функции протестируйте каждую рекурсивную функцию
Пример вывода для "Ханойских башен" (n=3): Переместить диск 1 с A на C Переместить диск 2 с A на B Переместить диск 1 с C на B Переместить диск 3 с A на C Переместить диск 1 с B на A Переместить диск 2 с B на C Переместить диск 1 с A на C

Комплексное задание: Система управления студентами ПРОДВИНУТЫЙ

Создайте программу для управления данными студентов, используя функции:

Структура данных:

struct Student { char name[50]; int id; int grades[5]; // оценки по 5 предметам double average; };

Требуемые функции:

  1. void addStudent(Student students[], int& count) - добавление нового студента
  2. void displayStudent(const Student& s) - вывод информации о студенте
  3. void displayAllStudents(const Student students[], int count) - вывод всех студентов
  4. void calculateAverage(Student& s) - вычисление среднего балла студента
  5. void calculateAllAverages(Student students[], int count) - вычисление средних баллов для всех студентов
  6. void sortByAverage(Student students[], int count) - сортировка студентов по среднему баллу
  7. void sortByName(Student students[], int count) - сортировка студентов по имени
  8. Student* findById(const Student students[], int count, int id) - поиск студента по ID
  9. Student* findByName(const Student students[], int count, const char* name) - поиск студента по имени
  10. void removeStudent(Student students[], int& count, int id) - удаление студента по ID
  11. void updateStudent(Student students[], int count, int id) - обновление данных студента
  12. void getTopStudents(const Student students[], int count, Student top[], int& topCount, int n) - получение n лучших студентов
  13. void getFailedStudents(const Student students[], int count, Student failed[], int& failedCount) - студенты с неудовлетворительными оценками
  14. void saveToFile(const Student students[], int count, const char* filename) - сохранение данных в файл
  15. void loadFromFile(Student students[], int& count, const char* filename) - загрузка данных из файла

Меню программы:

======================================== СИСТЕМА УПРАВЛЕНИЯ СТУДЕНТАМИ ======================================== 1. Добавить студента 2. Показать всех студентов 3. Найти студента по ID 4. Найти студента по имени 5. Удалить студента 6. Обновить данные студента 7. Показать лучших студентов 8. Показать неуспевающих студентов 9. Сортировать по среднему баллу 10. Сортировать по имени 11. Сохранить в файл 12. Загрузить из файла 0. Выход
Рекомендации:
  • Используйте std::string для работы со строками (подключите #include <string>)
  • Для ввода строк с пробелами используйте cin.getline() или getline()
  • При работе с файлами используйте #include <fstream>
  • Не забывайте проверять границы массивов
💡 Как выполнять задания:
1. Создайте новый файл с расширением .cpp для каждого задания
2. Напишите код, решающий задачу
3. Скомпилируйте программу (g++ имя_файла.cpp -o имя_программы)
4. Запустите программу и проверьте её работу
5. Сравните ваше решение с ответами для самопроверки
